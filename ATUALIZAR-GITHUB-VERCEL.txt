COMO ATUALIZAR O GITHUB

1. Extraia este ZIP.
2. Abra o repositório no GitHub.
3. Clique em Add file > Upload files.
4. Arraste TODO o conteúdo da pasta extraída para o GitHub.
5. Confirme em Commit changes.

- manifest.json atualizado para permitir Vercel/Render.
- backend/package.json atualizado.
- backend/api/[...path].js criado para rodar no Vercel.
- .gitignore criado para evitar envio de .env e node_modules.
