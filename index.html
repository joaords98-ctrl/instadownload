InstaDownload Pessoal

Salve links, baixe fotos/capas expostas e organize referências do Instagram direto na Vercel, sem backend.

v1.0 seguro
Para detectar foto/capa em qualidade exposta, abra o post no Instagram e use a extensão Chrome inclusa no pacote.

Importação da extensão

Quando a extensão detectar mídias na página aberta, ela pode enviar tudo para este painel.

Dica: suba este projeto no GitHub e publique na Vercel. Para rodar local: python -m http.server 5173.
