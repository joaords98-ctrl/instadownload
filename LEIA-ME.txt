INSTALAÇÃO NO CHROME

1. Extraia este ZIP.
2. Abra chrome://extensions.
3. Ative Modo do desenvolvedor.
4. Clique em Carregar sem compactação.
5. Selecione esta pasta extraída: ig-media-downloader-v04-carregar-direto.

IMPORTANTE:
O arquivo manifest.json agora está na raiz da pasta, então você pode selecionar a pasta principal.

BACKEND OPCIONAL PARA VÍDEOS/REELS VIA API OFICIAL

A pasta backend fica dentro desta mesma pasta.
Para rodar:

cd backend
npm install
npm start

Configure IG_USER_ID e IG_ACCESS_TOKEN antes de iniciar o backend.
